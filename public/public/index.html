Rexo Commands & Help

Prefix

You can use the prefix .. or slash commands (e.g., /help).

Quick Examples

  • ..help — Show this help menu with all commands
  • ..ping — Check bot latency
  • ..clear <amount> — Bulk delete messages (requires Manage Messages)
  • ..kick @user [reason] — Kick a member
  • ..ban @user [reason] — Ban a member
  • ..mute @user — Temporarily mute a member (timeout)
  • ..antilink on|off — Toggle anti-link protection
  • ..antispam set <max> <seconds> — Configure anti-spam rules
  • ..wordfilter add|remove|list <word> — Manage banned words
  • ..nukemode on|off — Enable/disable emergency safe mode
  • ..enable — Interactive security settings menu

Moderation

  • ..kick @user [reason] — Kick a member from your server.
  • ..ban @user [reason] — Ban a member.
  • ..mute @user — Temporarily mute a member. Use slash commands for duration.
  • ..unmute @user — Unmute a previously muted member.
  • ..clear <amount> — Bulk delete messages. Requires Manage Messages permission.

Security & Anti-Spam

  • ..antilink on|off — Toggle anti-link system. Whitelist certain channels if needed.
  • ..antispam set <max> <seconds> — Set max messages allowed per timeframe.
  • ..wordfilter add|remove|list <word> — Manage banned words in chat.
  • ..nukemode on|off — Emergency mode: automatically lock down channels if spamming detected.
  • ..enable — Opens interactive security menu to toggle all features.

Info & Utility

  • ..help — Show full command menu.
  • ..ping — Check bot latency.
  • ..uptime — Check how long the bot has been online.
  • ..serverinfo — Show server information.
  • ..roleinfo — Show info about a role.
  • ..userinfo @user — Show info about a member.

Warnings & Logs

  • ..warn @user [reason] — Issue a warning to a user.
  • ..warnings @user — Show warnings a user has.
  • ..softban @user [reason] — Kick + delete messages (temporary ban).
  • ..tempban @user [duration] — Temporarily ban a user.
