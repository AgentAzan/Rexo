Commands & Features

..help

Displays the help menu with all available commands in the server.

Usage: ..help or /help

Toggleable: No

..antilink

Toggle anti-link protection in your server. Automatically deletes messages containing links from non-whitelisted users.

Usage: ..antilink on | ..antilink off

Toggleable: Yes

..mute

Mute a member temporarily. Can be used with slash commands for specific durations.

Usage: ..mute @user

Toggleable: No
