          Core Commands & Features        

       
                             
           

..help / /help

           

Displays the comprehensive help menu and command documentation in the server.

           
             

                Usage: ..help | /help              

           
         
                   
           

..antilink

           

Toggle anti-link protection. Only links on the **internal whitelist** are allowed.

           
             

                Usage: ..antilink on|off              

           
         
                             
           

..mute / /timeout

           

Temporarily silence a member using Discord's native Timeout feature.

           
             

                Usage: ..mute @user 1h [reason]              

           
         
         
           

..warn

           

Issues a formal warning and logs it in the internal database for tracking.

           
             

                Usage: ..warn @user [reason]              

           
         
         
           

..warnings

           

View a member's full warning history stored in the bot's database.

           
             

                Usage: ..warnings @user              

           
         
         
           

..nukemode (Admin)

           

Manually activate or deactivate the **emergency safe mode** (anti-nuke system).

           
             

                Usage: ..nukemode on|off
